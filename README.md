# Ketcher

1. 项目背景
Ketcher 是由著名的软件开发商 EPAM Systems 开发并开源的。它是一款完全基于 Web（HTML5/JavaScript）的工具，不需要像老牌的 ChemDraw 那样安装客户端，在浏览器里就能画出出版级的化学图形。

2. 核心功能
可视化绘图：支持绘制单键、双键、环状结构（如苯环、环己烷）、立体化学键（楔形键等）。

原子和官能团：内置元素周期表，可以快速插入各种原子或复杂的官能团。

模板库：提供大量预设的模板，比如氨基酸、生物分子等。

化学格式支持：它可以识别并导出多种专业化学格式，如 Molfile (V2000/V3000)、SMILES、InChI 等。

反应式绘制：不仅能画分子，还能画带箭头的化学反应路径。

3. 与 MathQuill 的区别
MathQuill：主要处理“符号”，输出的是文本代码（LaTeX）。

Ketcher：主要处理“拓扑连接”，它理解分子之间的空间结构。它不仅是一个绘图工具，还是一个化学信息学（Cheminformatics）工具。

4. 为什么它在下载页提供 HTML 文件？
Ketcher 3.7.0 的部署非常友好：

单机运行：你下载的这个 HTML 版本通常是包含所有依赖的 Standalone（独立）版。

即插即用：就像你刚才折腾 MathQuill 一样，这个文件可以被直接丢进 IIS、Cloudflare Pages 甚至是直接用浏览器打开运行，它不依赖特定的后端服务器环境（除非你需要使用它的高级远程计算功能，如 Indigo 服务）。

5. 部署建议（结合你之前的需求）
上传到 CF Pages：你可以直接把解压后的 Ketcher 文件夹上传到 Cloudflare Pages，它会立即变成一个在线的专业化学实验室。

IIS 注意事项：Ketcher 可能会用到一些特殊的 JSON 或 XML 配置文件，在 IIS 部署时同样需要确保 MIME 类型 设置正确。

架构兼容性：Ketcher 的最新版本对现代浏览器（64位 Chrome/Edge）优化极好，由于它运算量比数学编辑器大，在较旧的 32位系统浏览器上运行可能会略有卡顿，但在目前的现代环境下表现非常稳定。

一句话总结： 如果你需要做一个化学考试系统或实验室管理系统，Ketcher 是目前市面上能找到的最好的免费、开源平替，功能完全不输给昂贵的商用版 ChemType。
